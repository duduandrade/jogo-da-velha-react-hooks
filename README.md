# Jogo da Velha
 Jogo da velha utilizando Node.js, React.js, Context e React Hooks (useState, useEffect e useContext)

## Requisitos
- NPM 8.11.0;

## Instalação

```bash
  cd jogodavelha-react-hooks
  npm install
```
